# PideYaApp — Entrega Parcial 2

## Descripción
Aplicación Android desarrollada en Kotlin con Jetpack Compose para la gestión simulada de pedidos de comida.

En esta segunda entrega se ha implementado la navegación completa entre pantallas y un nuevo tema visual basado en la gastronomía española.

## Funcionalidades
- Navegación Splash → Login → Registro → Home
- Sistema básico de autenticación
- Persistencia local de sesión
- Arquitectura MVVM
- Base de datos local con Room
- Tema visual personalizado (Material 3)

## Tecnologías
- Kotlin
- Jetpack Compose
- Navigation Compose
- Material 3
- Room
